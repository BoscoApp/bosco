# Roadmap

_Where Bosco is headed. Phases gate on completion, not calendar; each phase ends shippable._

## 🚧 In progress — v0.1.0 (Phase 0 — Decisions & skeleton)

**The skeleton that enforces the promises.** SvelteKit static scaffold, design-token system (both
theme hooks + liturgical colours), the versioned localStorage state module, the content schema with
`review_status` gating and the `archives[]` field, the 1962 liturgical calendar, the Portal shell
with a working reading-level switch, the external-URL CI guardrail, and the multi-arch Docker image.

**Done when:** `docker run` on an offline machine serves the styled Portal with working navigation
and a working tier switch; zero external requests confirmed by CI.

## 🗓️ Next — v0.2.0 (Phase 1 — Vertical slice)

The Library engine (tiered rendering, per-article override, cross-links, Pagefind search), **18
topics × 3 tiers** fully reviewed, the Field Guide index + card album, the Art Studio MVP (paint,
coloring mode, PNG export, **print button**), liturgical theming live on the Portal, and the **parent
trust page**. Board + epics stand up at kickoff.

**Done when:** a real 6-year-old and a real 10-year-old each spend 20 unassisted minutes and want to
come back; runs offline via Docker; parent trust page live.

## 🔭 Later — v0.3.0 (Phase 2 — Depth) · public-launch gate

Typing Trainer with faith/nature passage packs, two finished Arcade games, memory-work mode (Chapel
drills + progressive word-hiding), Chapel expansion (prayer book, calendar browser), the Seedling
pre-reader shell, named device-local profiles, the card album across destinations, PWA service
worker, the **Meadow** theme, and first Archives shelves. **Public launch at end of Phase 2 at the
earliest, with both themes live.**

## 🌠 Later — Phase 3 (Breadth & polish)

Content scale-up toward 100+ topics, recorded prayers/chant, print/PDF packs, The Sacristy, My Room
gallery, optional Kiwix shelf, accessibility + performance audit.
